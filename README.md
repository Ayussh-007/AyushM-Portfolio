## ✨ Cinematic Features

- **Immersive 3D Universe**: A dynamic WebGL background featuring evolving abstract monoliths, volumetric fog, and an interactive grid.
- **Fibonacci Technology Sphere**: A mathematically balanced, camera-facing 3D skill constellation built with React Three Fiber.
- **Kinetic Experience Architecture**: Massive brutalist typography with character-based motion-blur reveals and tri-planar depth parallax.
- **Project Universes**: Each project is an independent interactive simulation—ranging from AI soundwaves to neural geometric grids.
- **Ultra-Premium Micro-Interactions**: Velocity-sensitive liquid cursor, elastic magnetic physics, and holographic reflection effects.
- **Luxury Theme Transformation**: A seamless world-changing transition system between high-contrast Dark and airy Light modes.
- **Performance Optimized**: Locked 60 FPS experience powered by hardware acceleration, MotionValues, and adaptive quality scaling.

## 🛠️ Technical Stack

- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4
- **Animation**: GSAP + Framer Motion
- **3D Engine**: Three.js (React Three Fiber + Drei)
- **Smooth Scroll**: Lenis
- **UI Components**: Shadcn UI + Custom SVG Filters

## 🚀 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ayussh-007/AyushM-Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📦 Deployment

This project is optimized for **Vercel**. Connect your GitHub repository to Vercel and it will automatically detect the Next.js settings and deploy your cinematic world.
